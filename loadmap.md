# Sound-Learn 12주 개발 로드맵

기준일: 2026-03-30  
시험기간 제외: 2026-04-13 ~ 2026-04-27  
개발 인원 구성: 프론트엔드 2명, 백엔드 2명  
공동 작업: UI 설계, AI 활용, 테스트, 문서화, 발표 준비

## 프로젝트 목표
사용자의 가창 음성을 녹음 또는 업로드 받아, 원곡 MIDI와 비교하여 음정, 박자, 음역대를 분석하고, 이를 시각화 및 AI 피드백 형태로 제공하는 웹 기반 보컬 분석 서비스를 완성한다.

## 전체 개발 흐름
1. 음성 파일 업로드 및 녹음 기능 구현
2. 오디오 파일에서 주파수 추출
3. MIDI 기준곡 데이터 처리
4. 사용자 음성과 기준곡 비교
5. 분석 결과 시각화
6. AI 피드백 생성
7. 테스트, 배포, 발표 준비
